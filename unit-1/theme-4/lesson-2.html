Создание контроллера

Для взаимодействия с моделью, нам нужен контроллер, создадим его с помощью следущей команды: 

php artisan make:controller NewUserController --resource

В данном случае --resource сразу добавляет в контроллер нужные методы. Наш контроллер находится в папке app/Http/Controllers.

Добавление маршрута

Теперь добавим следущий маршрут в файл web.php: 

use App\Http\Controllers\NewUserController;
                            
Route::resource('users', NewUserController::class);

В данном случае --resource сразу добавляет в контроллер нужные методы. Наш контроллер находится в папке app/Http/Controllers.

Создание представлений

Перейдем к созданию шаблонов. Для этого в папке resources/views создаем папку users где и будут хранится наши шаблоны.

Для начала создадим основу, в которую будет вставляться наш контент, назвав его layout.blade.php, она будет иметь следующий код: 

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>CRUD операции</title>
</head>
<body>
    <div class="container">
        @yield('content')
    </div>
</body>
</html>

Вывод результатов создания пользователя будет в index.blade.php: 

@extends('users.layout')
@section('content')

<h2>Добавление нового пользователя</h2>

@if ($message = Session::get('success'))
    <div class="alert alert-success">
        <p>{{ $message }}</p>
    </div>
@endif
@endsection

Создание пользователя расположится в create.blade.php: 

@extends('users.layout')
@section('content')

<h2>Добавление нового пользователя</h2>

@if ($errors->any())
    <div class="alert alert-danger">
        <ul>
            @foreach ($errors->all() as $error)
                <li>{{ $error }}</li>
            @endforeach
        </ul>
    </div>
@endif

<form action="{{ route('users.store') }}" method="POST">
    @csrf


        <strong>Имя:</strong>
        <input type="text" name="name" placeholder="Иван">
        <strong>Пароль:</strong>
        <input type="password" name="password">
        <button type="submit">Создать</button>

    </div>
</form>
@endsection

И наконец отображение пользователей будет в show.blade.php: 

@extends('users.layout')
@section('content')

<h2>Просмотр пользователей</h2>
@foreach ($users as $user)
<div>
    <strong>Имя:</strong>
    {{ $user->name }}
    <strong>Пароль:</strong>
    {{ $user->password }}
</div>
@endforeach

Настройка контроллера

Теперь произведем настройку контроллера следущим образом: 

<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;
use App\Models\User;

class NewUserController extends Controller
{
    /**
        * Display a listing of the resource.
        */
    public function index()
    {
        return view('users.index');
    }

    /**
        * Show the form for creating a new resource.
        */
    public function create()
    {
        return view('users.create');
    }

    /**
        * Store a newly created resource in storage.
        */
    public function store(Request $request)
    {
        $request->validate([
            'name' => 'required',
            'password' => 'required',
        ]);
        User::create($request->all());
        return redirect()->route('users.index')->with('success','Пользователь добавлен успешно.');
    }

    /**
        * Display the specified resource.
        */
    public function show(string $id)
    {
        $users = User::all();
        return view('users.show', compact('users'));
    }

}

Теперь при переходе по users/create вы попадаете на страницу создания пользователя. После его создания он попадет в вашу таблицу. А при переходе по users/show будут выведены все пользователи из таблицы.
