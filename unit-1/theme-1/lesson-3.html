Установка программ

Open Server

Для запуска проекта на Laravel понадобиться локальный сервер, в нашем случае это Open Server. Скачать его можно по ссыле https://ospanel.io/download/. При установке настройки можно оставить по умолчанию.

После установки у вас появится иконка красного флажка, на которую нужно нажать:

Далее мы нажимаеи на "Папка с проектами". Должна открыться папка domains, уже там нужно создать папку с названием вашего проекта, в моем случае это будет "Laravel-Course".

Отлично! Теперь нужно вернуться к предыдущему меню и перейти в Настройки->Домены. Проверьте что в настройке Управление доменами стоит настройка Автопоиск доменов.

После переходи в "Модули" и выставляем настройки на на скриншоте. HTTP: Apache_2.4-PHP_8.0-8.1.

Настройка Open Server завершилась!

Visual Studio Code

Теперь нужно установить редактор кода. Для этого мы будем использовать Visual Studio Code, просто скачайте его по ссылке: https://code.visualstudio.com/download.

Все нужные программы на данный момент установлены, можно приступать к установке Laravel.

Создание проекта Laravel

Установка Laravel

Для начала нам нужно сделать следующее: запустить Open Server и открыть консоль.

После в консоли нужно перейти в папку с проектом с помощью команды cd domains/название-папки-проекта. А затем использовать следующую команду: 

composer create-project laravel/laravel .

После этого произойдет установка файлов Laravel в папку с вашим проектом.

Базовая настройка

Мы установили Laravel в папку с нашим проектом, а теперь нам нужно сделать так, чтобы при переходе на наш домен(название папки) мы попадали на главную страницу. Для этого открываем настройки Open Server->Домены и выбираем в настройке "Автопоиск корневой папки домена" последний пункт.

Теперь, для того чтобы проверить работоспособность проекта, нажмите на иконку Open Server->Мои проекты->Ваш проект.
